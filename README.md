# Art Recreate Portfolio

Simple React portfolio experience for Ramya Kovida's Art Recreate brand. The layout focuses on clean sections, responsive cards, and lightweight styling.

## Key Components

- Responsive hero, gallery, class, workshop, and contact sections.
- Lightweight gallery cards built with plain React and CSS.

## Scripts

- `npm install`
- `npm run dev`
- `npm run build`
- `npm run preview`
